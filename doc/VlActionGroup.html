VlActionGroup()

Toon meerdere knoppen of links. De groep zorgt ervoor dat ze correct zijn uitgelijnd.

Constructor

new VlActionGroup()

Properties:
Name Type Description
align string Attribuut wordt gebruikt om ervoor te zorgen dat de onderliggende elementen worden gealigneerd. Mogelijkheden: align="center" of align="right".
space-between boolean Attribuut wordt gebruikt om aan te duiden dat de ruimte tussen de elementen volledig moet worden opgevuld.
bordered boolean Attribuut wordt gebruikt om aan te duiden dat de tussenliggende elementen een rand krijgen.
