VlButton()

Gebruik de vl-button om een CTA toe te voegen. Het type call-to-action wordt bepaald door het label of de pictogram.

Constructor

new VlButton()

Properties:
Name Type Description
data-vl-disabled boolean Attribuut wordt gebruikt om aan de gebruiker aan te duiden dat de functionaliteit niet actief is.
data-vl-error boolean Attribuut wordt gebruikt om het belang of de gevolgen van een actie te benadrukken.
data-vl-block boolean Attribuut wordt gebruikt om ervoor te zorgen dat de knop getoond wordt als een block element en bijgevol de breedte van de parent zal aannemen.
data-vl-large boolean Attribuut wordt gebruikt om de aandacht van de gebruiker te trekken door de font-size te vergroten.
data-vl-wide boolean Attribuut zorgt ervoor dat de knop breder op het scherm zal getoond worden.
data-vl-narrow boolean Attribuut zorgt ervoor dat de knop smaller op het scherm zal getoond worden.
data-vl-loading boolean Attribuut wordt gebruikt om aan de gebruiker aan te geven dat zijn actie momenteel verwerkt wordt.
data-vl-secondary boolean Attribuut wordt gebruikt in combinatie met een gewone knop om alternatieve acties te voorzien.
data-vl-tertiary boolean Attribuut wordt gebruikt in combinatie met gewone en secondary knoppen om alternatieve acties te voorzien.
