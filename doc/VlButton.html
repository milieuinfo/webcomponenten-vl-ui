Class: VlButton

VlButton()

Gebruik de vl-button om een ​​call-to-action toe te voegen. Het type call-to-action wordt bepaald door het label of de pictogram.

Constructor

new VlButton()

Properties:
Name Type Description
disabled boolean Attribuut wordt gebruikt om aan de gebruiker aan te duiden dat de functionaliteit niet actief is.
error boolean Attribuut wordt gebruikt om het belang of de gevolgen van een actie te benadrukken.
block boolean Attribuut wordt gebruikt om ervoor te zorgen dat de knop getoond wordt als een block element en bijgevol de breedte van de parent zal aannemen.
large boolean Attribuut wordt gebruikt om de aandacht van de gebruiker te trekken door de font-size te vergroten.
wide boolean Attribuut zorgt ervoor dat de knop breder op het scherm zal getoond worden.
narrow boolean Attribuut zorgt ervoor dat de knop smaller op het scherm zal getoond worden.
loading boolean Attribuut wordt gebruikt om aan de gebruiker aan te geven dat zijn actie momenteel verwerkt wordt.
secondary boolean Attribuut wordt gebruikt in combinatie met een gewone knop om alternatieve acties te voorzien.
tertiary boolean Attribuut wordt gebruikt in combinatie met gewone en secondary knoppen om alternatieve acties te voorzien.
