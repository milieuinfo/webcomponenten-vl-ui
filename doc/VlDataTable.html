Class: VlDataTable

VlDataTable()

Gebruik een data table om op een gestructureerde manier (grote hoeveelheden) relationele data te tonen. Demo.

Constructor

new VlDataTable()

Properties:
Name Type Description
hover boolean Attribuut wordt gebruikt om een rij te highlighten waneer de gebruiker erover hovert met muiscursor.
matrix boolean Attribuut wordt gebruikt om data in 2 dimensies te tonen. Zowel de rijen als de kolommen krijgen een titel. Deze titels worden gescheiden door een dikke lijn.
lined boolean Variant met een lijn tussen elke rij en kolom.
zebra boolean Variant waarin de rijen afwisslend een andere achtergrondkleur krijgen. Dit maakt de tabel makkelijker leesbaar.
