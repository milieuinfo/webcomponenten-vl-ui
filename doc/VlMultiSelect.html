Class: VlMultiSelect

VlMultiSelect()

Gebruik een multiselect om een gebruiker toe te laten om in een lijst van vooropgestelde keuzes te zoeken, en enkele of alle passende keuzes te selecteren.

Constructor

new VlMultiSelect()

Properties:
Name Type Description
block boolean Attribuut wordt gebruikt om ervoor te zorgen dat de textarea getoond wordt als een block element en bijgevolg de breedte van de parent zal aannemen.
error boolean Attribuut wordt gebruikt om aan te duiden dat het select element verplicht is of ongeldige tekst bevat.
success boolean Attribuut wordt gebruikt om aan te duiden dat het select element correct werd ingevuld.
disabled boolean Attribuut wordt gebruikt om te voorkomen dat de gebruiker iets kan kiezen uit het select element.
