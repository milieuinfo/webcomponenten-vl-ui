new VlPager()

Description

Gebruik de pager component om het aantal beschikbare pagina's weer te geven, markeer de huidige pagina en voeg navigatie knoppen toe.

Properties
Name Type Description
data-vl-total-items number

Attribuut wordt gebruikt om totaal van elementen te bepalen.
data-vl-current-page number

Attribuut wordt gebruikt om huidige pagina te bepalen.
data-vl-items-per-page number

Attribuut wordt gebruikt om het aantal rijen per pagina te bepalen.
data-vl-pagination-disabled number

Attribuut wordt gebruikt om geen pagina links te tonen.
data-vl-align-center boolean

Attribuut wordt gebruikt om de paginatie te centreren.
data-vl-align-right boolean

Attribuut wordt gebruikt om de paginatie rechts uit te lijnen.
