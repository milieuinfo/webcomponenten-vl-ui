Class: VlRichDataTable

VlRichDataTable()

Een tabel op basis van een dynamische lijst van data die uitgebreid kan worden met functionaliteiten die het consumeren van de data door een gebruiker kunnen verbeteren.

Constructor

new VlRichDataTable()

Properties:
Name Type Description
data-vl-data string De data die door de tabel getoond moet worden in JSON formaat.
data-vl-collapsed-m boolean Vanaf een medium schermgrootte zullen de cellen van elke rij onder elkaar ipv naast elkaar getoond worden.
data-vl-collapsed-s boolean Vanaf een small schermgrootte zullen de cellen van elke rij onder elkaar ipv naast elkaar getoond worden.
data-vl-collapsed-xs boolean Vanaf een extra small schermgrootte zullen de cellen van elke rij onder elkaar ipv naast elkaar getoond worden.
data-vl-multisort boolean Laat de gebruiker sorteren op meer dan 1 kolom.
data-vl-filter-closable boolean Attribuut dat de filter sluitbaar maakt en een knop getoond wordt om de filter te tonen en terug te verbergen. Op een klein scherm wordt een modal geopend bij het klikken op de filter knop ipv een de filter naast de tabel te tonen. Om elementen van de filter te verbergen enkel in de modal, kan het attribuut data-vl-hidden-in-modal gezet worden.
data-vl-filter-closed boolean Attribuut dat aangeeft of dat de filter gesloten is.
