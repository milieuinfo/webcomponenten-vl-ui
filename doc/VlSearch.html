VlSearch

  1. Home
  2. Classes
  3. VlSearch

new VlSearch()

Description

Gebruik de vl-search component zodat een gebruiker zoekcriteria kan ingeven om specifieke content te vinden.

Properties
Name Type Attributes Default Description
data-vl-label string <optional>
 'Zoekterm'

Attribuut wordt gebruikt als label voor zoekcriteria.
data-vl-submit-label string <optional>
 'Zoeken'

Attribuut wordt gebruikt als label voor de submit knop.
data-vl-block boolean

Attribuut duidt aan dat een breed zoekveld met knop wordt getoond. Dit is de standaardweergave.
data-vl-inline boolean

Attribuut duidt aan dat een smal zoekveld met kleine knop wordt gebruikt.
data-vl-alt boolean

Attribuut bepaalt of de alternatieve weergave (witte achtergrond) wordt gebruikt. Alleen relevant in combinatie met data-vl-block.
Examples
Breedte aanpassen:
			<style>
			    :root {
			      --vl-search-width: 50%;
			    }
			   </style>
