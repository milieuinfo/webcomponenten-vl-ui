Class: VlSidesheet

VlSidesheet()

Side-sheet zijn containers die aan de linker- of rechterrand van het scherm zijn verankerd.

Constructor

new VlSidesheet()

Properties:
Name Type Description
left boolean Attribute wordt gebruikt om aan te duiden dat de side-sheet de linkererand van het scherm moet plaatsen.
right boolean Attribute wordt gebruikt om aan te duiden dat de side-sheet de rechterkant van het scherm moet plaatsen.
enable-swipe boolean Attribute wordt gebruikt om aan te duiden dat swipe functie toegelaten is.

Example
Breedte van de side sheet aanpassen(op grote scherm):
 static get styles() {
    return [
      css`
        :host {
         --vl-side-sheet-width: 480px;
         }
        `
    ];
  }
